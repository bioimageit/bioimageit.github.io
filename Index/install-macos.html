macOS install

To install BioImageIT on MacOS (COMPATIBLE WITH INTEL MACS, WE WOULD APPRECIATE SOME FEEDBACK IF YOU HAVE A M1 MAC), download the installer from here


Double click BioImageIT_install.dmg to open it


Double click BioImageIT_install to start the install (it will be installed by default in username/BioImageIT)


When the install is finished, BioImageIT is available in the Launchpad
